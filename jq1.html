Short Selectors Examples

Id Selector in JavaScript

document.getElementById("idname")

jQuery Selector

${"#idname"}

Class Selector in JavaScript

document.getElementsByClassName("classname")

jQuery Selector

${".classname"}

Tag Selector in JavaScript

document.getElementsByTagName("tagname")

jQuery Selector

${"tagname"}
